Обязательно реализовать:

9. INVARIANTS (НЕ ЛОМАТЬ)
1. navigation не зависит от data
2. каждый экран имеет back
3. структура меню фиксирована
4. нет лишних кнопок
5. нет дублирующих переходов
10. EDGE CASES

11. READY CHECK

Модуль готов если:

1. есть полный путь START → CARD
2. можно вернуться назад с любого экрана
3. нет тупиков
4. меню не ломается
5. переходы стабильны
